## 📋 What is PatternGen

PatternGen helps you make animated designs for videos. You create a layout in Figma and then use this tool to add moving patterns behind your text. The software turns these designs into flat images that you load into 3D software like Blender. This process creates high-quality motion graphics for title sequences.

## 🛠 Features

*   **Figma Integration:** You export your artboards directly from your design software. The tool reads your color choices and text positions automatically.
*   **Procedural Animation:** You choose patterns that grow and move across your screen. You do not need to draw each frame by hand.
*   **Layered Output:** The system creates image files with transparent backgrounds. This makes it easy to stack images in your video editor.
*   **Blender Ready:** The exported files work with 3D planes to create a depth effect in your scene.

## 🚀 Setting Up Your Project

After you open the app, you need to prepare your design.

1. **Open Figma:** Create your title design. Make sure your layers have names that match the scene requirements.
2. **Export Data:** Use the export plugin to save your scene as a JSON file. This file contains the location of your text and the colors you chose.
3. **Load in App:** Open the PatternGen app and select the Open File option. Pick the JSON file you just saved.
4. **Select Pattern:** Pick an animation style from the list on the left.
5. **Adjust Parameters:** Change the speed, size, and color of the pattern. Watch the preview window to see changes in real time.
6. **Generate Sequence:** Click the Export button to save your files as a sequence of PNG images.

## ⚙️ Troubleshooting

If the program fails to open, ensure your Windows system is up to date. Sometimes security software might flag new programs. Click More Info and then Run Anyway if you trust the source. If the preview window stays black, update your computer graphics drivers. Close other programs that require intensive graphics power while you render your image sequences.
